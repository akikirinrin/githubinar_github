# GitHubinar of GitHub

## コンセプト
GitHub学習の手引きを執筆してもらう
理由: 座学よりも、もっと効率のよい学習方法があるから
- 実際に手を動かしながら
- これから学ぼうとしている人に教える
これらを満たすのが、テキストの執筆だ！
